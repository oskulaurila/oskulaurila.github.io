
# Software Development Skills: Front-End 2022-23

Front-End course project of a mockup portfolio site.

## Installation

Install node-sass

```bash
  npm install node-sass
  npm run sass
```
    
## Deployment

To deploy this project run

```bash
  npm run deploy
```
